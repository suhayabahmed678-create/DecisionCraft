#  DecisionCraft – Personal Decision Scoring Engine

DecisionCraft is a Python-based Object-Oriented decision-making tool that helps users
compare multiple options logically instead of emotionally.

It evaluates decisions based on:
- Pros & Cons
- Importance
- Risk level

Then calculates a normalized score and recommends the best option.

---

Features

- Object-Oriented Design (OOP)
- Multiple decision comparison
- Score normalization (0–100)
- Input validation & error handling
- Clear recommendation output
- Beginner-friendly but professional structure

---

Technologies Used

- Python 3.x
- Standard Python libraries only (no external dependencies)
